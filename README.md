# zmb-fractal-registration

Fractal tasks for stitching and registration of OME-Zarr images.

## Installation

### Local installation
Clone the repository:

```bash
git clone https://github.com/ZMB-UZH/zmb-fractal-registration
cd zmb-fractal-registration
```

And install either via conda & pip:

```bash
conda create -n zmb-fractal-registration
conda activate zmb-fractal-registration
pip install .
```

Or pixi:

```bash
pixi install
```

### Installation on Fractal server
To install this task on a fractal server, use the .whl file from the latest release.
